(오늘은 반복문, 컴포넌트 여러개, props(★),  watch 배움)

중복되는 부분은 컴포넌트로 만들어주는것이 좋음

LottoGenerator 에서 
<lotto-ball v-for="ball in winBalls" :key="ball.key"></lotto-ball>
이 태그를 통해 lottoBall.vue를 사용하고 있음
그럼 LottoGenerator가 부모가 되고, lottoBall이 자식이됨
부모 -> 자식으로 데이터를 넘겨줄 수 있음

ex)
 <lotto-ball v-for="ball in winBalls" :key="ball.key" number="5"></lotto-ball>
 이상태면 number 5를 전달
즉, props는 부모컴포넌트가 자식컴포넌트에게 데이터를 전달

LottoGenerator(숫자를 뽑는 역할)
LottoBall(데이터를 표시하는 역할)

올바른 방법은 데이터를 바꾸는 부모의 메소드(함수)를 자식에게 props로 전달하는것

축약형
props: ['number'],

 export default {
        components: {
            'lotto-ball':LottoBall, 
        },
        이거를
           components: {
            LottoBall,  //실무에서 많이 사용하는 방법
        },
        이렇게 줄이기 가능
