*****3-3
검색바의 가로세로 높이는 border를 포함한 길이

 box-sizing: content-box; : box-sizing의 기본
 >>border / padding / content 다따로함
>>    box-sizing: border-box;로 바꾸면 3개다 포함됨
>>with를 border기준으로 바꾼것
>> 이걸 안쓰면 숫자를 다 계산하면서 해야함

왠만하면 모든 태그에 border-box를 적용해주는게 좋음
>>
*{
    box-sizing: border-box;
}

브라우저마다 기본 css가 다름
>>다 없에주는게 좋음(css reset)
>> 기본 브라우저 css파일이 따로 있음

#header-search fieldset input{
    border:none;
}
>>input태그의 border선을 지워줌

그래도 input클릭시 파랑색 테투디가 남아있음(outline)
