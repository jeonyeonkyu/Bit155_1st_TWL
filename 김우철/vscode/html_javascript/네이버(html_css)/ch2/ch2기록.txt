div를 백퍼센트차지하지않고 줄여야됨(width를 달아주자)

자식 + 손자 = 자손

#header-search > h1 //자식
#header-search h1 //자식이 아닌 모든것(손자, 증손자 등)

보통은 자손선택을 많이함(꺽쇠 쓰기 귀찮아서)

css는 선택자개념을 이해하고 css 속성은 암기해야함

#header-search h2{
    display:none;
} >>스크린 리더도 못읽음


html의 기본인 display:block은 전체 가로줄 너비를 다 차지함
display:inline-block;으로 컨텐츠 크기만큼 줄여줄 수 있다(margin 제거)

숙달이 되면 html태그로도 매치를 예상할 수 있어야 함

글자만 있으면 span으로 묶어서 써라
span은 기본 display가 인라인임(그냥 인라인은 width를 지정하지 못함 / 자기 콘텐츠 크기만 가질 수 있음)
